## "Student Registration & Academic Management System"

*This project is a web-based student registration and academic management system designed for a university environment.
It allows administrators to manage student records and academic information, while students can view their profiles and generate academic reports.
The project and focuses on frontend development with HTML and CSS, frontend interactivity with JavaScript,
backend logic with PHP, database interaction and secure data handling using MySQL.*

##  Features
**Admin Functionality**
-Register new students
-View a list of all registered students
-View individual student profiles
-Update student information
-Delete student records (with confirmation prompt)
-Secure database operations using prepared statements

**Student Functionality**
-View personal academic profile
-Generate and view reports:
  =Profile Summary Report
  =Registration Confirmation Slip

**Reports**
-Auto-generated profile summary
-Registration confirmation with status and timestamps
-PDF generation using FPDF
-Printable / downloadable format

**Technologies Used**
-Frontend:HTML, CSS, JavaScript
-Backend:PHP, FPDF (PDF report generation)
-Database:MySQL

**Security & Validation**
-Input validation using PHP string functions
-Error handling for form submissions
-SQL injection prevention using prepared statements
-Confirmation dialogs for destructive actions (delete)
